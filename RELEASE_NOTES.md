# 年报抓取桌面版 Release Notes

## 版本信息

- 产品名称：年报抓取桌面版
- 版本号：4.1.0
- 发布日期：2026-05-09
- 项目主页：<https://github.com/Little-pig-create/AnnualReportSpider>

## 本版重点更新

- 统一消息提示为 `Element Plus Notification`（可自动消失）
- 执行任务按钮点击即触发通知（提交中 / 启动成功 / 启动失败）
- 新增任务完成通知能力（系统通知 + 可选提示音）
- 新增历史任务“一键复跑（带原配置快照）”
- 新增历史任务“手动刷新提示”（开始/成功/失败）
- 顶部作者信息卡片增加轻量呼吸灯视觉效果
- 前端按需分包与懒加载优化，构建体积进一步下降
- 前后端错误码语义统一（桥接层统一映射）
- README 全面重构，补齐开发、运行、打包与发布说明

## 兼容性说明

- 推荐系统：Windows 10 / 11
- 需要 WebView2 Runtime（大多数 Windows 系统已内置）
- 前端需要 Node.js 18+ 用于构建

## 升级建议

1. 先备份旧配置与输出目录
2. 拉取最新代码后执行前端重新构建：`cd webui && npm install && npm run build`
3. 如需安装包分发，按 `PACKAGING.md` 重新打包
